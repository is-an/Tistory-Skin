# 🌈 Tistory-Skin: Modern Portfolio with Claude AI


> **"Crafting digital experiences with code & AI."**
>
> 티스토리 블로그를 현대적인 IT 포트폴리오로 변신시켜주는 프리미엄 스킨입니다. 글래스모피즘(Glassmorphism) 디자인과 클로드 AI 어시스턴트 위젯이 통합되어 있습니다.

---

### ✨ Key Features

- 💎 **Glassmorphism Design**: 반투명 블러 효과와 세련된 그라데이션이 적용된 현대적인 UI.
- 🤖 **Claude AI Widget**: 방문자와 소통하는 스마트 AI 챗봇 기능을 기본 탑재.
- 📱 **Fully Responsive**: 데스크탑부터 모바일까지 완벽하게 대응하는 반응형 레이아웃.
- 🌗 **Dark/Light Mode**: 사용자 취향에 맞춘 부드러운 테마 전환 기능.
- 🚀 **Performance Optimized**: 부드러운 스크롤 애니메이션과 최적화된 로딩 속도.

### 🚀 Quick Start

1. **파일 다운로드**: `tistory-skin/` 폴더 내의 파일들을 확인합니다.
2. **티스토리 적용**: 
   - 티스토리 관리자 > 꾸미기 > 스킨 변경 > 스킨 등록
   - `index.xml`, `skin.html`, `style.css`, `scripts.js`를 업로드합니다.
3. **AI 설정**: `scripts.js` 파일에서 `ANTHROPIC_API_KEY`를 자신의 키로 변경하면 AI 챗봇이 활성화됩니다.
